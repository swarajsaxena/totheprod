## Usage

```tsx

export const HomePage = () => {
  return (
    <TtpBlurFocusNavigation
      brand='My Brand'
      items={[
        { label: 'Home', href: '/' },
        { label: 'About', href: '/about' },
        { label: 'Services', href: '/services' },
        { label: 'Contact', href: '/contact' },
      ]}
    />
  )
}
```

## Examples

### Basic Usage

```tsx
<TtpBlurFocusNavigation
  brand='My Brand'
  items={[
    { label: 'Home', href: '/' },
    { label: 'About', href: '/about' },
  ]}
/>
```

### Custom Brand Element

```tsx
<TtpBlurFocusNavigation
  brand={
    <span className="bg-gradient-to-r from-blue-500 to-purple-500 bg-clip-text text-transparent">
      Brand
    </span>
  }
  items={[...]}
/>
```

### Without Blur

```tsx
<TtpBlurFocusNavigation
  brand="My Brand"
  items={[...]}
  showBlur={false}
/>
```

### Custom Blur Intensity

```tsx
<TtpBlurFocusNavigation
  brand="My Brand"
  items={[...]}
  blurIntensity={20}
  backgroundOpacity={0.9}
/>
```

### Non-Sticky Nav

```tsx
<TtpBlurFocusNavigation
  brand="My Brand"
  items={[...]}
  sticky={false}
/>
```

## Styling

The component uses Tailwind CSS and respects your theme configuration. The blur effect uses CSS `backdrop-filter` for optimal performance.

## Best Practices

- Use the blur effect for modern, glassmorphic designs
- Adjust `backgroundOpacity` based on your content contrast needs
- Consider using `sticky={false}` for full-page layouts
- Customize the brand element to match your design system
