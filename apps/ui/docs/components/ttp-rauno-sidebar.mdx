## Customization

You can customize the sidebar using Tailwind classes or by modifying the component directly.

### Example with Custom Styling

```tsx
<RaunoSidebar
  className="bg-primary text-primary-foreground"
  open={isOpen}
  onToggle={(state) => setIsOpen(state)}
/>
```

## Best Practices

> **Tip**: Keep the sidebar content minimal for better mobile experience.

- Use clear labels for navigation items
- Implement keyboard shortcuts for power users
- Consider adding tooltips for collapsed state
