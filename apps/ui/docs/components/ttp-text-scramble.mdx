# Text Scramble

An animated text component that creates a scrambling effect where characters cycle through random characters before revealing the original text, perfect for attention-grabbing headings and dynamic text reveals.

## Examples

### Basic Usage

```tsx
<TextScramble text="Hello World!" />
```

### Custom Duration

```tsx
<TextScramble text="Fast Scramble" duration={1000} />
```

### With Trigger Control

```tsx
const [trigger, setTrigger] = useState(false)

return (
  <>
    <TextScramble
      text="Click to reveal"
      trigger={trigger}
      onComplete={() => console.log('Done!')}
    />
    <button onClick={() => setTrigger(true)}>Start Animation</button>
  </>
)
```

### Custom Styling

```tsx
<TextScramble text="Styled Text" className="text-4xl font-bold" charClassName="text-blue-500" />
```

## How It Works

1. Each character starts hidden
2. Characters reveal sequentially from left to right
3. Each character cycles through random characters before settling on the final character
4. The scrambling creates a decryption-like effect

## Accessibility

- Maintains semantic HTML structure
- Text remains readable by screen readers
- No motion interference with assistive technologies
