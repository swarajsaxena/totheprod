# Horizontal Text Reveal

A scroll-driven text animation component that reveals text word-by-word with elegant horizontal sliding motion as the user scrolls through the page.

## Description

The Horizontal Text Reveal component creates an engaging scroll-based animation where text slides in from the right and fades in word by word. Each word animates sequentially based on scroll progress, creating a smooth and dynamic reading experience. Perfect for hero sections, storytelling elements, and attention-grabbing headlines.


```bash
npx shadcn add ui.totheprod.com/r/ttp-horizontal-text-reveal.json
```

## Usage

### Basic Example

```tsx
import { HorizontalTextReveal } from '@/components/ui/totheprod-ui/ttp-horizontal-text-reveal'

export const HeroSection = () => {
  return (
    <div id="scroll-container" className="h-screen overflow-y-auto">
      <div className="min-h-[300vh]">
        <div className="sticky top-1/2 -translate-y-1/2">
          <HorizontalTextReveal
            text="Your amazing headline goes here and reveals as you scroll"
            scrollContainerId="scroll-container"
          />
        </div>
      </div>
    </div>
  )
}
```

### With Custom Styling

```tsx
<HorizontalTextReveal
  text="Custom styled text with larger font"
  scrollContainerId="scroll-container"
  className="text-6xl font-bold text-blue-600"
/>
```

### With Custom Overlap

```tsx
<HorizontalTextReveal
  text="Words will overlap more during animation"
  scrollContainerId="scroll-container"
  overlap={0.5}
/>
```

## Examples

### Hero Section with Long Text

```tsx
export const Hero = () => {
  const text =
    "What's on my mind? Mostly the quiet tragedy of watching humans outsource their thinking to glowing rectangles."

  return (
    <div id="hero-scroll" className="h-screen overflow-y-auto">
      <div className="min-h-[400vh]">
        <div className="sticky top-1/2 -translate-y-1/2 px-8">
          <HorizontalTextReveal
            text={text}
            scrollContainerId="hero-scroll"
            className="text-5xl font-medium leading-tight"
          />
        </div>
      </div>
    </div>
  )
}
```

### Faster Reveal with Less Overlap

```tsx
<HorizontalTextReveal
  text="Quick reveal with minimal overlap"
  scrollContainerId="scroll-container"
  overlap={0.1}
  className="text-4xl"
/>
```

## Important Notes

1. **Scroll Container**: The component requires a scrollable container with an ID. Make sure to set `overflow-y-auto` on the container.

2. **Minimum Height**: Use `min-h-[300vh]` or larger on the wrapper to create enough scroll distance for the animation.

3. **Sticky Positioning**: Use `sticky top-1/2 -translate-y-1/2` to keep the text centered while scrolling.

4. **Overlap Value**: Lower values (0.1-0.2) create snappier reveals, higher values (0.4-0.6) create smoother, more flowing animations.
